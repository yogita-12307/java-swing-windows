# java-swing-windows

A beginner-friendly Java Swing project demonstrating how to create and navigate between multiple GUI windows.
This project focuses on event-driven programming, JFrame management, and basic GUI design using Java Swing.

## Features

- Create multiple windows (`JFrame`) in Java Swing
- Navigate between windows using button clicks
- Use `FlowLayout` and `BorderLayout` for layout management
- Display messages using `JLabel` with custom fonts
- Simple and readable code ideal for beginners
